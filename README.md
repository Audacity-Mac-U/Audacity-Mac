## Overview

Audacity Professional Suite is a complete open-source audio production platform that serves content creators and audio professionals with the plugin ecosystem, automation capability, and extensibility needed for production-scale audio work without commercial software costs. It extends the core editing tools with plugin support, scripting automation, and community-developed extensions that expand its capabilities continuously.

Plugin support hosts VST, AU, LV2, and Nyquist effect plugins that extend the built-in effects library with any third-party audio processing tool. The plugin ecosystem includes thousands of free effects, analyzers, and generators from the open-source audio community. Nyquist, Audacity's built-in scripting language, enables creation of custom effects and analysis tools without external development tools.

Spectral editing displays audio as a frequency spectrogram and enables selection and editing of specific frequency ranges, making it possible to surgically remove or modify sounds at specific frequencies without affecting other frequency content. The spectral selection tools handle tasks like removing tonal noise, isolating specific instruments, and repairing frequency-specific problems that waveform-only editing cannot address.

Macro automation records and replays sequences of editing operations, enabling consistent processing across multiple files. Macros can be applied to entire folders of audio files through batch processing, converting manual editing workflows into automated production pipelines. The macro system handles format conversion, level normalization, noise reduction, and any other processing chain that needs to be applied consistently across many files.

Audacity Professional Suite is the audio production platform for users who need capable, extensible audio tools without commercial licensing costs, backed by a global open-source community that ensures continuous development and improvement. Its zero-cost model and plugin ecosystem make professional audio production accessible to everyone regardless of budget.

## Performance & Integration

Audacity is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Audacity processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
